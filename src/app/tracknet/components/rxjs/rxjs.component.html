    • Difference b/w mergemap,switchmap,concatmap

    Use mergeMap if you simply want to flatten the data into one Observable, use switchMap if you need to flatten the data into one Observable but only need the latest value and use concatMap if you need to flatten the data into one Observable and the order is important to you.

    forkjoin() and combineLatest()

    forkJoin - When all observables complete, emit the last emitted value from each. Observable and the order is important to you.

    combineLatest - When any observable emits a value, emit the latest value from each. Usage is pretty similar, but you shouldn't forget to unsubscribe from combineLatest unlike forkJoin.
